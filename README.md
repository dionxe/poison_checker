Rails application for job application. Includes some tests for models

# Get started
```sh
$ git clone https://github.com/dafontaine/poison_checker
$ cd poison_checker
$ bundle install
$ rails db:setup
$ rails server
```

# Run tests
```sh
$ rails test
```

# Requirements
- Ruby 2.5
- Bundler
- SQLite 3
